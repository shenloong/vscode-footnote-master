# 脚注大师

提供轻松便捷的脚注操控体验，让文档注释整理事半功倍。

## 特性

- 提供便捷的脚注上标代码生成功能，允许自定义代码样式。
- 能够快速将带有序号的脚注文本转换为规范的脚注列表项。
- 实现快速调整脚注序号的功能，节省开发者的时间和精力。

## 用法

- 生成脚注上标代码：

  通过使用快捷键 `Alt+S` 或右键菜单选择 `脚注大师 > 生成脚注上标代码` 运行指令。

  - 未选序号：生成空的上标代码 `<sup data-footnote=""></sup>`，供手动输入序号。
  - 选中序号：生成包含选中序号的上标代码 `<sup data-footnote="序号">序号</sup>`。

- 生成脚注列表项代码：

  将 `序号. 脚注文本。` 转换为 `<li id="footnote-序号">脚注文本。</li>`

  1. 选中文案文档中的脚注列，并将其复制到代码编辑器中。
  2. 选中包含脚注的文本行，通过使用快捷键 `Ctrl+Alt+F` 或右键菜单选择 `脚注大师 > 生成脚注列表项代码` 运行指令。
  3. 操作完成后，会保留每个列表项中的文本换行，同时清除冗余的空行和因复制单元格内换行文本产生的额外双引号。

- 脚注序号重新排序：

  当文案的脚注部分有增删变动时，需要从变更位置开始，依次调整所有后续脚注的序号。

  1. 通过使用快捷键 `Ctrl+Alt+R` 或右键菜单选择 `脚注大师 > 脚注序号重新排序` 运行指令。
  2. 输入脚注起始序号和排序规则，按回车键执行排序操作。
  3. 完成后，所有匹配的脚注上标和列表项序号都将自动更新。
